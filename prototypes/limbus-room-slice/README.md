# 림부스 비주얼 룸 슬라이스

ACT1 맵 노드 `act1-room-002`, `act1-room-052`, `act1-room-003`, `act1-room-006`을 3개의 횡스크롤 플레이 화면으로 변환한 기준 샘플이다.

- 기준 해상도: 1280×720
- 월드 폭: 3840px, 3화면
- 진행 방향: 추락 지점(오른쪽) → 고철 평원 → 교량 출구(왼쪽)
- 배경: 각 방 전용 생성 이미지 3장
- 충돌: 배경 이미지와 독립된 임시 콜라이더 레이어
- 플레이어: `lento-vertical-slice`의 기존 티크 액션별 128×128 프레임 직접 재사용
- 애니메이션: 대기, 달리기, 점프, 공격, 대시를 상태별로 분리 재생
- 비율 규칙: 균일 스케일과 바닥 중앙 피벗만 사용

정적 서버에서 `index.html`을 실행한다. 개발자용 충돌 박스는 `H`로 전환한다.

## Unity 이전 기준

- 배경 이미지는 방별 Sprite 또는 Parallax Layer로 분리한다.
- 지형 충돌은 배경과 분리된 Tilemap Collider 또는 Composite Collider로 구현한다.
- 티크 프레임은 Pivot을 Bottom Center로 통일한다.
- 캐릭터에는 비균일 스케일을 적용하지 않는다.
- 배경 생성 이미지는 최종 픽셀 아트 제작의 구도·재질 레퍼런스로도 사용할 수 있다.

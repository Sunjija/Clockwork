# 렌토 전투 버티컬 슬라이스

렌토 보스전의 패턴 전달력과 전투 흐름을 검증하는 1280x720 웹 프로토타입이다. 640x360 시뮬레이션 좌표를 2배 렌더 스케일로 출력해 전투 수치는 유지한다. 티크의 핵심 동작은 PixelLab 생성 프레임을 동작별 시트로 분리했고, 모든 프레임은 128x128 셀과 하단 중앙 피벗을 사용한다. 렌토 패턴은 원본 콘셉트 보드에서 분리한 독립 PNG를 사용한다. 정화시설은 바닥에 맞닿은 원형 관로 중심의 픽셀아트 배경으로 구성했다.

## 실행

`index.html`을 브라우저에서 직접 열 수 있다. 키보드 포커스가 빠진 경우 게임 화면을 한 번 클릭한다.

| 입력 | 기능 |
|---|---|
| `←` / `→` | 이동 |
| `Z` | 점프 |
| `X` | 기본 공격 |
| `C` | 대시 |
| `↑` | 배관 숨기 / 제어반 조작 |

오른쪽 검증 패널에서 페이즈 이동, 패턴 강제 실행, 무적, 히트박스, 전조 속도와 시간 배율을 조절할 수 있다. 실험용 패링은 원 기획에 확정되지 않은 기능이므로 기본값이 꺼져 있다.

## 검증 순서

1. 1페이즈에서 배관 가까이 공격해 소리를 내고, 느린 물기 뒤 공격 창이 읽히는지 확인한다.
2. 렌토 체력 70% 이하에서 새끼쥐 우선 처리가 자연스럽게 전달되는지 확인한다.
3. 꼬리 휩쓸기를 점프로 피할 수 있는지, 피격 시 납치가 지나치게 길지 않은지 확인한다.
4. 독 피격 뒤 이동 둔화가 위험하지만 회복 가능한 수준인지 확인한다.
5. 20% 이하에서 낙하 위험 지대와 3연속 음파의 간격을 확인한다.
6. 돌진 전조 중 배관 가까이에서 `↑`를 눌러 숨고, 렌토의 벽 충돌 그로기를 만든다.
7. 처치 후 오른쪽 제어반에서 `↑`를 눌러 배수를 완료한다.

## 파일

- `data/lento-combat.json`: 전투 수치와 페이즈 데이터 초안
- `data/animation-manifest.json`: 티크·렌토·새끼쥐 애니메이션 제작 목록
- `data/action-assets.json`: 할로우 나이트 키 배치와 동작별 독립 PNG 경로
- `PIXELLAB_GENERATION_BRIEF.md`: PixelLab 참조 고정 문구, 프레임 수, 안전 여백
- `LENTO_VISUAL_AND_COMBAT_SPEC.md`: 비주얼, 전조, 판정 원칙
- `UNITY_HANDOFF.md`: Unity 이관 기준
- `reference/tique-character-sheet.png`: 사용자가 제공한 티크 원본 시트 사본

## 범위

이 빌드는 전투 판독성과 패턴 루프 검증용이다. 티크의 대기 4프레임, 달리기 8프레임, 점프 7프레임, 수평 공격 6프레임, 대시 6프레임, 피격 6프레임이 웹 런타임에 적용되어 있다. 위·아래 공격과 렌토 모션의 추가 프레임, 정식 사운드, 최종 밸런스는 후속 교체 대상이다.

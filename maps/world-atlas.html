전도에서 읽히는 것

3막의 동선 기하
ACT1 수직(내려갔다 올라옴) → ACT2 수평(도시 횡단) → ACT3 방사(축을 중심으로 회전). 같은 메트로베니아 문법으로 세 번 다른 기하를 통과한다.
물의 완전 순환
남탑 회수 → 배출 기부 → (도시·기초 관통, 점선) → 대공동 폭포 → 심층 습지. 튜토리얼에서 옆을 지났던 폭포의 수원을 최종막에서 위에서 내려다본다 — 세계의 물이 어디서 와서 어디로 가는지가 게임 전체에 걸쳐 완결.
티크의 원환
옥상에서 도시 안으로 추락 → 청소 로봇이 도심 밀폐 투입구에 티크를 넣음 → 지하 자동 수거관 → 림부스 각성 → 사냥꾼·기억② → 폐쇄 인양로 → B23 터미너스 → 서부 내벽 화물 승강장 → 도시 횡단 → 타워 등반 → 같은 옥상의 말파스.
ACT1 선택 확장
갈고리 재방문: 구 폐기물 선별장 4방 + 폐쇄 송수로 6방. 더블점프 재방문: 폐쇄 굴착지 6방. 세 지역 모두 먼저 잠긴 입구를 보여주고, 내부 루프 끝에서 MOD 재료 또는 MOD 3택1을 지급한 뒤 입구로 되돌아오는 숏컷을 연다.
폐기선·화물선 분리
티크 폐기선: 성벽 안쪽 도심 투입구 → 지하 자동 수거관 → 림부스. 청소 로봇은 성벽을 넘지 않는다. 화물선: 서부 내벽 화물 승강장 → 직하 터미너스 → B23 굴착 현장. 외곽 재활용 처리장은 별도 선택 지역이며 두 선과 직접 연결되지 않는다.
중앙 수직선
옥상(말파스·클레아) → 정제로 → 승강로 → 기슭 → 오수 기둥 → 대공동(렌토의 탄생지). 최종 보스와 첫 보스가 같은 수직선 위에 있다 — 타워의 빛과 그림자.

각 ACT의 크리티컬 패스·게이트 상세는 개별 문서(clockwork-map-detailed / act2 / act3) 참조. 본 전도는 열람·발표용 포스터 겸 인게임 전체 지도의 참조 원본.

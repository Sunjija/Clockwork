A. 실제 플레이 흐름 수정안 반영

터미너스 승강축 → 서부 내벽 화물 승강장[CP] → 서부 변두리 시장 → 공급 허브 지상부 → 메인 타워 잠금 목격 → 중앙 대광장 → 동부 중심 연결가 → 동부 빈 집 지대 → 들개형 기계 → 기억A + 벽타기
→ 동부 성벽 등반 → 외벽 지붕길 → 엘리아스 외벽 공방[CP] → 공방 관리자 키 → 동부 정비 수직구 → 배관 루프 → 정비 기계 → 기억B + 공급 복구 → 공급 허브 상승 리프트 → 메인 타워 수문장 → 과부하 반격 → ACT3

벽타기 획득 전에는 서부 내벽 승강장부터 들개형 기계까지 모든 필수 방이 성벽 안에 있다. 청소 로봇은 동부 빈 집 지대의 도심 밀폐 투입구까지만 이동하며, 티크는 여기서 지하 자동 수거관을 타고 림부스로 운반됐다. 로봇이 성벽을 넘은 적은 없다.

B. 원형 도시와 타워 연결 표현 규칙

중앙 메인 타워
ACT2의 최종 잠금이자 ACT3 중앙 허브. 중심부에 홀로 서며 네 방향 위성 타워를 묶는 릴레이 축이다.
네 방향 타워 위치
동·서·남·북 타워 기부는 성벽이나 외곽이 아니라 변두리(인지 1/2)와 중심(인지 1/4) 사이의 환상 경계에 같은 반지름으로 배치된다.
서부 타워 기부
물류 가도와 연결. 횡단면에 직접 보이며 ACT3 서탑 루트가 열리면 지상문도 개방된다.
동부 타워 기부
원석 반입·정비 가도와 연결. 서부 내벽 화물 승강장에서 들어온 화물은 성벽 안쪽 지하 순환 화물선을 거쳐 동탑 반입 설비로 전달된다.
북부·남부 타워 기부
횡스크롤 단면에서 깊이축을 그릴 수 없으므로 북부는 배경 방향, 남부는 전경 방향 분기 방으로 표시한다. 평면도에서는 실제 북·남 위치를 확인할 수 있다.
원형 성벽
서·동 보호벽은 별도 구조물이 아니라 하나의 거대한 원형 성벽의 양쪽 단면이다. 벽타기를 얻으면 양쪽 성벽 상부와 외곽 순환 탐사가 동시에 열린다.
티크 폐기 경로
도심 청소 로봇 → 성벽 안쪽 밀폐 투입구 → 지하 자동 수거관 → 림부스 추락 지점. 외곽 재활용 처리장은 플레이어가 벽타기 후 방문하는 별도 설비이며 티크 투입 사건과 무관하다.

C. 설정 정합성

문서 명시: 들개형 기계의 기억A·유대, 기계 보스와 기억코어의 결합 원칙, 시내에서 외곽으로 갈수록 인지가 강해지는 구배.

이번 구조화 제안: 원형 성벽, 네 방향 타워의 지상 기부, 성벽 내장형 화물 승강장, 북·남 가도 분기, 벽타기 후 양쪽 외곽 동시 개방.

결정 필요: 네 위성 타워 지상문을 ACT3 진행 중 역방향 숏컷으로 실제 개방할지, 외형과 복선만 제공하고 끝까지 봉쇄할지. 현재 맵 데이터는 ACT3 각 루트의 개방 플래그를 받는 문으로 임시 정의했다.

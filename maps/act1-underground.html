확정 9개 지역을 24개 룸으로 세분화한 제안입니다. 방위는 세계 단면 기준으로 확정: 동측(오른쪽)=림부스·터미너스(도시 상업지구 직하), 서측=칼리고·정수장치. 터미너스는 위아래로 뚫린 수직 원통 동굴(등반 전투 건틀릿). 확정 동선(①~⑨)과 문서의 확정 요소(보스·기억·보물방 3곳·오수 기둥)는 그대로 두고, 룸 분할·숏컷·게이트 배치는 전부 [제안]입니다.

A. 메인 크리티컬 패스 확정 동선의 룸 전개

추락 지점(폐기 슈트 종점 · 각성 · 봉인 개구부 목격) → 고철 평원(보물방 A — 정체불명 부품 3택1) → 다리 상부(쥐떼) → 다리 하부 → 쓰러짐(모르비 구조 · 암전)
→ 공방에서 눈뜸(수리 · 부품 판명 = MOD 튜토리얼 · 의뢰) → 광장[CP]
→ 낙하 샤프트▼일방향 → 대공동[CP] → 심층 문 전실(잠김 목격 = 자물쇠 먼저 ①)
→ 서측 통로 → 여과실 → 펌프실(밸브) → 중앙 정수조 렌토(내부 둥지 · 시설 정지) → 격파 → 티크가 재가동(배수 재개, 로그 한 줄) → 갈고리 획득
→ 취수구(보물방 B, 갈고리 즉시 사용) → 시험 홀 → 상승 챌린지 → 코어실 기억①
→ 대공동 2차(수위 하강, 문 개방) → 배수 수직갱 → 진흙 동굴(황금 파편 = 기억② 복선)
→ 침수 입구[CP] → 흔적 지대(끌린 자국 · 환경 예고) → 둥지 사냥꾼(매복+끌어당김 단일 전투) → 더블점프 + 기억 회수실 기억②
→ 터미너스 하단 개구부 → 하부 등반(거미떼 1파, G+D 종합 시험) → 화물 적재장[CP](보물방 C)
→ 상부 등반(거미떼 2파·붕괴 발판·거미줄) → 정상부 거대 거미 → 봉인 개구부 개방 → 지상

보스 직전마다 체크포인트 1개, 능력 획득 직후 30초 안에 그 능력을 쓰는 방(취수구/시험 홀, 하부 등반)을 배치해 "얻자마자 써먹는" 리듬을 만들었어.

B. 숏컷 · 게이트 설계 [제안]

낙하 샤프트 ▼
첫 통과는 일방향 낙하(칼리고→교차로). 갈고리 획득 후 역행 가능 — 기억① 회수 후 칼리고로 30초 복귀, 모르비 보고·개조 제작 왕복이 쾌적해짐. 파밍 편의가 아니라 순수한 교통 개선 — 메트로베니아식 숏컷.
심층 문 (GAT-101)
능력 게이트가 아닌 이벤트 게이트. 열쇠 = 렌토 격파 후 시설 재가동(배수 재개). "자물쇠 먼저" 원칙의 대표 사례로, 1차 방문 때 물에 잠긴 문 너머로 어둠 속 반짝임(황금 파편)을 살짝 보여줌.
취수구 G (EDG-013)
렌토 격파 직후 같은 방에서 갈고리로만 닿는 상단 통풍구 — 보물방 B. 새 능력의 첫 사용을 튜토리얼이 아닌 보상으로 처리.
송수관 (제안)
펌프실(B23)에서 정화수 저장소(B21)로 물을 끌어올리는 수직 관로 — 칼리고 생존 경제 2축(정화수)의 물리적 구현. 렌토 격파 전에는 침묵한 관, 격파 후에는 물 흐르는 소리·진동이 관을 따라 올라감. 교차로 수위 하강과 함께 "정수장치가 살아났다"를 두 곳에서 동시에 보여주는 환경 피드백.
봉인 개구부 (B20)
터미너스 정상부는 림부스 바로 옆 — 게임 시작 1분 안에 거미줄과 화물 잔해로 봉인된 최종 출구를 목격하게 됨. "자물쇠 먼저" 3호이자 가장 큰 스케일의 복선: 처음 본 그 문을 3시간 뒤 안쪽에서 연다. 격파 후에는 림부스↔터미너스 상시 숏컷.
터미너스 진입 G+D
하부 등반 구간이 갈고리+더블점프 복합 게이트 그 자체 — 별도의 잠금 없이 지형이 곧 자물쇠. 기억② 회수실에서 하단 개구부로 바로 이어져 보스 직후 동선이 끊기지 않음.
은닉 선반 (D)
정상부 둥지 옆, 거대 거미 격파 후 지상으로 나가기 직전 더블점프로 닿는 선반 — ACT1 마지막 빌드 보강. "시작점(림부스)을 반대편 높이에서 내려다보는" 순환의 마침표.

C. 터미너스 — 수직 전투 건틀릿 [제안]

위아래로 뚫린 원통형 동굴. 과거 기계 운반 터널이었기에 벽면에 정비용 발판과 갈고리 앵커 링이 남아 있다는 세계관 명분. 상단은 지상(0F), 하단은 지하심층 수면으로 관통 — 오른쪽 오수 기둥(추락)과 좌우 대칭을 이루는 "올라가는 축".

전투 방식
멈춰 서는 아레나 없음 — 등반하면서 싸움. 거미떼가 벽·천장·거미줄에서 연속 스폰되고, 발판이 좁아 제자리 방어보다 갈고리 이탈·회피 기동이 유리하게 설계 (방어 액션의 구체 시스템은 미정). 이동 능력 2종의 종합 시험장.
거미 생태 2단
거미떼는 결정화 이전부터 터미너스에 서식한 점진 변이 선주민(핵 없음 — 8-5 정합). 봉인 개구부·지형을 이루는 거미줄은 수년간 누적된 것. 거대 거미는 결정화 후 폐렌티움 폭증으로 그중 한 개체가 급성 거대화한 결과(핵 보유) — 타임라인 압박 해소.
낙하 페널티
추락 = 사망이 아니라 아래 세그먼트로 낙하(진행 손실 + 해당 구간 적 리팝). 최하단은 진흙 습지라 서사적으로도 안전망이 됨. 죽음 대신 "다시 올라가라"로 긴장 유지.
4단 리듬
하부 등반(학습: 고정 발판+앵커) → 화물 적재장(휴식: CP·보물방 C — 원통 벽에 박힌 옛 화물 플랫폼) → 상부 등반(심화: 붕괴형 발판·끈끈이 거미줄·2파) → 정상부 보스.
거대 거미 보스전
룰북 3-C 제안과 통합: 보스가 실 중심부(고지대)로 후퇴하는 패턴 — 갈고리로 접근하고 더블점프로 마지막 한 칸을 쫓는다. 원통 벽면 선회 + 페이즈 전환 시 발판층 봉쇄로 갈고리+더블점프 요구(GAT-003)를 전투 안에서도 구현. 격파 시 정상부 봉인이 뜯기며 빛이 쏟아져 들어오는 연출 → 지상.

D. 체크포인트 배치 [제안 — 시스템 미정]

광장(마을) / 대공동(허브) / 침수 입구(심층 진입) / 화물 적재장(건틀릿 중간·최종전 앞) — 보스 전마다 1개, 코너당 1개 원칙. 정체(벤치·충전소)와 사망 페널티는 미정이라 위치만 잡아둔 상태.

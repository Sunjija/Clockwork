A. 구조 개념 — 타워 평면 = 시계판 [제안]

십자 5탑
위성탑 4개(동·서·남·북)가 시계판의 12·3·6·9시 위치, 메인 타워가 축. 각 위성탑은 렌티움 순환의 한 단계를 담당 — 반입(동) → 정제(메인) → 방출(서) / 공급 → 회수(남) → 직하 배출(기부). 타워 단지 전체가 렌티움의 순환 회로이자 거대한 시계.
회전 가교 = 시곗바늘
중층 허브의 단일 가교는 시곗바늘처럼 회전해 한 번에 한 위성탑만 접속한다. 금색 실선은 현재 접속, 금색 점선은 다음 자세, 청회색 점선은 화면 깊이축의 북·남 자세다. 동탑 하층으로 가는 구리색 이중선은 가교가 아니라 고정 화물 통로다.
수직 순환의 완성
배출 기부가 오수 낙하 기둥(RGN-008)의 상류 — ACT1에서 맞았던 폭포의 수원을 ACT3에서 위에서 내려다보게 됨. 남탑 회수 계통 → 배출 기부 → 대공동 폭포 → 심층. 세계의 물이 어디서 와서 어디로 가는지가 3막에 걸쳐 완결.

B. 진행 사슬 초안 [제안]

기슭(ACT2 종점) → 하층 로비[CP](중앙 승강로 봉쇄 목격 — 자물쇠 먼저) → 원석 하역장 → 화물 통로 → 동탑 기부
→ 반입 권양층(과부하 반격 릴레이) → 크레인 정상 → 가교 동력 확보 → 회전 가교 허브[CP] 가동
→ 가교를 서탑 자세로 회전 → 방출 제어층 → 서탑 수문장 → 짧은 공중 대시 + 권한 코어 A(대시 귀환 · 남탑 자세 해금)
→ 가교 남편: 남탑 → 회수 집수층 → 남탑 수문장 → 권한 코어 B(RGN-008 상류 목격 · 북탑 자세 해금)
→ 가교 북편: 기록 서고 → 열람층 → 기존 기억 A·B와 기록을 대조해 기억 재해석 + 감사 서명(새 기억코어는 지급하지 않음 · RUL-003 공개)
→ 과부하 반격 + 공중 대시 + 동력 + 코어 A+B + 감사 서명 → 중앙 승강로 개방 → 정제로 코어 → 옥상: 말파스(프롤로그의 그 옥상 — 원환 완성)

"자물쇠 먼저" 계승: 승강로 봉쇄를 1분 안에 목격 → 위성탑 3~4개를 돌아 열쇠를 모아 돌아온다. ACT1(수직 하강→상승), ACT2(수평 횡단)에 이어 ACT3는 방사형(축을 중심으로 돌기) — 3막의 동선 기하가 전부 다름.

C. 각 탑의 기능과 서사 [제안]

동탑 — 반입
원석 반입 크레인. 원석은 서부 내벽 화물 승강장·터미너스에서 성벽 안쪽 지하 순환 화물선을 거쳐 동탑 기부로 전달된다. 림부스 수거관과는 설비를 공유하지 않는다.
서탑 — 방출
대기 방출 노즐(RUL-011b ①). 도시의 '시간'이 뿜어져 나오는 곳 — 결정화 사건 때 최대 출력으로 쓰인 바로 그 계통.
남탑 — 회수
배관망 회수 계통 집결(RUL-011b 보강안). 폐렌티움이 모여 배출 기부로 떨어짐 — ACT1 전체를 만든 원인의 제어실.
북탑 — 기록
[제안 · 필수 경로] 엘리아스의 연구 기록동. 렌티움 발명(RUL-002)과 은폐(RUL-003)의 문서를 기억 A·B의 장면과 대조한다. 새 기억코어를 더 주지 않고, 이미 본 기억의 의미를 뒤집어 말파스 대면의 동기를 완성하며 중앙 승강로용 감사 서명을 얻는다.
옥상 — 말파스
확정 앵커. 클레아 보존 거처(RUL-008)이자 티크가 던져진 곳(EVT-009). 기억①의 "내려다보는 시점" 복선이 여기서 회수(4장) — 보스전 무대 자체가 복선의 답.

D. 미정 — 다음에 정할 것

흐름 검증을 위해 서탑·남탑 수문장, 서탑의 짧은 공중 대시, 북탑 필수화와 6조건 승강로를 [제안]으로 임시 반영함. ACT3 보스는 서탑 수문장·남탑 수문장·말파스의 3기다. 최종 결정이 필요한 것은 ① 공중 대시와 과부하 반격의 세부 수치 ② 두 수문장의 이름·외형·패턴 ③ 북탑 RUL-003의 구체 문서와 기억 재해석 연출 ④ 가교 회전 조작 방식 ⑤ 말파스전 페이즈 구조.

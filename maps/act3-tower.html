A. 구조 개념 — 타워 평면 = 시계판 [제안]

십자 5탑
위성탑 4개(동·서·남·북)가 시계판의 12·3·6·9시 위치, 메인 타워가 축. 각 위성탑은 렌티움 순환의 한 단계를 담당 — 반입(동) → 정제(메인) → 방출(서) / 공급 → 회수(남) → 직하 배출(기부). 타워 단지 전체가 렌티움의 순환 회로이자 거대한 시계.
회전 가교 = 시곗바늘
중층 허브의 가교는 시곗바늘처럼 회전해 접속 탑을 바꾼다. 3-9시 자세면 동·서탑, 12-6시 자세면 남·북탑(화면 깊이축) 접속. 횡스크롤에서 남북 = 깊이 문제를 "다리를 돌려서 간다"로 해결 — 건너는 순간은 화면 전환 하나면 되므로 구현 비용도 낮음. 가교 회전 자체가 퍼즐/게이트 장치(자세 전환 권한·동력).
수직 순환의 완성
배출 기부가 오수 낙하 기둥(RGN-008)의 상류 — ACT1에서 맞았던 폭포의 수원을 ACT3에서 위에서 내려다보게 됨. 남탑 회수 계통 → 배출 기부 → 대공동 폭포 → 심층. 세계의 물이 어디서 와서 어디로 가는지가 3막에 걸쳐 완결.

B. 진행 사슬 초안 [제안]

기슭(ACT2 종점) → 하층 로비[CP](중앙 승강로 봉쇄 목격 — 자물쇠 먼저) → 원석 하역장 → 화물 통로 → 동탑 기부
→ 반입 권양층(수직 등반) → 크레인 정상 → 가교 동력 확보 → 회전 가교 허브[CP] 가동
→ 가교 3-9시: 서탑 → 방출 제어층 → 노즐 상부 → 권한 코어 A
→ 가교 12-6시(회전): 남탑 → 회수 집수층 → 회수 제어 → 권한 코어 B(RGN-008 상류 목격)
→ 북탑: 기록 서고 → 열람층 진실 기록(RUL-003 공개 후보 — 필수 여부·시점 미정)
→ 코어 A+B → 중앙 승강로 개방 → 정제로 코어 → 옥상: 말파스(프롤로그의 그 옥상 — 원환 완성)

"자물쇠 먼저" 계승: 승강로 봉쇄를 1분 안에 목격 → 위성탑 3~4개를 돌아 열쇠를 모아 돌아온다. ACT1(수직 하강→상승), ACT2(수평 횡단)에 이어 ACT3는 방사형(축을 중심으로 돌기) — 3막의 동선 기하가 전부 다름.

C. 각 탑의 기능과 서사 [제안]

동탑 — 반입
원석 반입 크레인. 도시 물류 축(동측 — 화물 터미널·터미너스·림부스)의 지상 종점이 타워까지 이어짐. 통합 단면에서 동쪽 수직선이 하나로 완성.
서탑 — 방출
대기 방출 노즐(RUL-011b ①). 도시의 '시간'이 뿜어져 나오는 곳 — 결정화 사건 때 최대 출력으로 쓰인 바로 그 계통.
남탑 — 회수
배관망 회수 계통 집결(RUL-011b 보강안). 폐렌티움이 모여 배출 기부로 떨어짐 — ACT1 전체를 만든 원인의 제어실.
북탑 — 기록
엘리아스의 연구 기록동. 렌티움 발명(RUL-002)과 은폐(RUL-003)의 문서가 잠들어 있음 — 진실 공개(시점 미정, 1-B)의 유력 후보 지점. 필수 경로로 할지 선택 탐색으로 할지는 공개 시점 결정에 종속.
옥상 — 말파스
확정 앵커. 클레아 보존 거처(RUL-008)이자 티크가 던져진 곳(EVT-009). 기억①의 "내려다보는 시점" 복선이 여기서 회수(4장) — 보스전 무대 자체가 복선의 답.

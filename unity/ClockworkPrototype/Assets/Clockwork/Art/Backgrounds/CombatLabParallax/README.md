# Combat Lab Parallax Background

Prototype-only generated art for the `CombatLab` scene. These images are visual experiments, not setting canon.

## Layers

- `combat-lab-far.png`: opaque distant cavern and collapsed industrial settlement.
- `combat-lab-mid.png`: transparent maintenance structures, pipes, and suspended machinery. Regenerated with a green hard key to preserve violet-blue metal colors.
- `combat-lab-near.png`: retired foreground experiment retained for comparison; it is not instantiated in `CombatLab`.

All three files are 1280 x 720 pixel-art layers. The active scene uses only the far and middle layers. The current middle source was generated against a flat green key, converted with a hard matte, and nearest-neighbor downscaled.

## Generation Brief

- Strict side-view industrial underground environment.
- Charcoal, cold blue-gray steel, muted dark teal, oxidized brass, and sparse amber light.
- Deliberate chunky pixel clusters with no text, UI, characters, or bright toxic green.
- Far: lowest contrast and quiet central corridor.
- Mid: separated structures with generous gaps and no continuous floor.
- Near: optional room-specific framing only; excluded from the combat lab for readability.

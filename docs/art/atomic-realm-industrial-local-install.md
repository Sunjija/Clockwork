# Atomic Realm Industrial Tileset Local Install

The Atomic Realm Industrial Tileset permits commercial use and modification but prohibits redistribution. Its source files and generated Unity assets must remain under the Git-ignored local folder:

`unity/ClockworkPrototype/Assets/Clockwork/LocalAssets/AtomicRealm/IndustrialTilesetFree`

The prototype builder uses the local Atomic Realm floor and platform tiles when they are installed. If they are absent, it falls back to the tracked CLOCKWORK prototype tiles.

Source: https://atomicrealm.itch.io/industrial-tileset

# 티크 인게임 모델 고정 규칙

이 문서는 동작 이미지가 달라져도 티크가 같은 캐릭터로 보이게 하는 외형 기준이다. 포즈보다 아래 규칙을 우선한다.

## 핵심 실루엣

- 머리는 전체 불투명 높이의 약 50%를 차지하는 큰 구형이다. 동작 중 타원형으로 찌그러뜨리지 않는다.
- 몸통은 머리보다 좁은 둥근 형태이며 전체 높이의 약 34%를 차지한다.
- 다리는 몸통 바로 아래 붙는 매우 짧은 2단 관절이다. 발을 포함한 노출 높이는 전체 높이의 16%를 넘지 않는다.
- 양발은 기본 자세에서 몸통 가로폭 안쪽에 놓인다. 공격 준비 자세에서도 다리를 길게 늘려 키를 키우지 않는다.
- 팔은 짧은 분절형이다. 공격 시 팔을 펴더라도 각 관절의 굵기와 기본 길이는 유지한다.
- 등 뒤 태엽 키는 인게임 모델에서 제거한다.

## 고정 랜드마크

| 항목 | 고정 기준 |
|---|---|
| 소스 캔버스 | 640×512 투명 PNG |
| 소스 피벗 | `(320, 480)` |
| 접지선 | 발바닥 `y=480` |
| 머리 | 구형, 프레임 간 폭·높이 변화 3% 이내 |
| 몸통 | 원형에 가까운 형태, 프레임 간 크기 변화 4% 이내 |
| 다리 | 2단 관절, 노출 높이 16% 이하 |
| 눈 | 시안 타원형 2개, 크기와 간격 고정 |
| 하트 코어 | 몸통 중앙, 크기 변화 4% 이내 |
| 표시 배율 | 모든 동작 공통 `0.34` |

## 팔레트 기준

- 색상 기준 이미지는 `generated/v3/idle/idle-01-settle.png`이다.
- 황동 중간색 기준은 약 `#BC783B`, 시안 발광 중심 기준은 약 `#0CF3FC`로 잡는다.
- 새 동작은 기준 이미지의 밝기 분포에 맞춘 뒤 사용한다. 포즈 참고 이미지의 더 노랗거나 더 밝은 금속색을 그대로 가져오지 않는다.
- 눈과 하트 코어의 시안색만 발광색으로 취급한다. 황동 몸체 전체에 시안 반사를 추가하지 않는다.

## 동작 제작 금지사항

- 동작을 강조하기 위해 캐릭터 전체를 확대하거나 축소하지 않는다.
- 다리를 길게 만들어 공격 자세의 키를 높이지 않는다.
- 머리와 몸통을 가로·세로로 따로 늘리지 않는다.
- 포즈마다 황동색, 시안 발광색, 외곽선 농도를 새로 해석하지 않는다.
- 대기 모션에서 서로 크기가 다른 이미지를 교차 재생하지 않는다.

## 대기 모션

- 대기는 동일한 기준 이미지 한 장을 사용한다.
- 발과 전체 배율은 고정한다.
- 움직임은 최대 1px의 세로 이동과 하트 코어의 미세한 밝기 변화로만 표현한다.
- 호흡처럼 보이게 만들기 위해 머리나 몸통을 확대·축소하지 않는다.

## 팔 비율 잠금

- 팔은 어깨를 포함해 최대 3절로 유지한다.
- 팔 한 절의 굵기는 머리 가로폭의 18%를 넘지 않는다.
- 주먹 지름은 머리 가로폭의 20%를 넘지 않는다.
- 완전히 뻗은 공격 팔도 몸통 가장자리부터 주먹 끝까지 머리 가로폭의 55%를 넘지 않는다.
- 걷기와 대시의 후방 팔도 공격 팔보다 길거나 굵어지지 않는다.

## 공격·대시 실루엣 잠금

- 공격과 대시는 기준 대기 이미지의 머리, 몸통, 팔, 다리 픽셀 비율을 그대로 사용한다.
- 공격과 대시 중 머리 크기 및 팔·다리 길이 편차는 0으로 고정한다.
- 동작감은 캐릭터 부위 확대·축소 없이 전신 회전, 위치 이동, 타격 효과와 속도선으로 표현한다.

# CLOCKWORK — 아트 합류 제안 브리프

> 2D 메트로베니아 「클락워크」 팀에서 아트 담당을 찾고 있습니다.
> 이 문서 하나로 프로젝트의 그림, 확정된 작업 스펙, 그리고 합류 시 맡게 될 일을 파악할 수 있게 정리했습니다.

---

## 1. 어떤 게임인가

- **제목:** 클락워크 (CLOCKWORK)
- **장르:** 2D 횡스크롤 메트로베니아 — 할로우나이트·데드셀류를 좋아하는 플레이어가 타깃
- **플랫폼:** Steam (PC)
- **분량:** 1차 범위 ACT1(지하) 약 3시간 → 본편 약 10시간(ACT1·2·3)
- **출시 목표:** 2027 겨울 ~ 2028 봄
- **팀:** 기획·스토리·연출 1인 + Unity 개발 1인 — 여기에 **아트 1인**이 합류하면 3인 체제가 완성됩니다

**한 줄 컨셉:** 시간이 멈춘 도시. 황금빛으로 굳어버린 사람들. 기억을 잃은 조수 로봇 '티크'가 지하 밑바닥에서 깨어나, 도시가 멈춘 날의 진실을 향해 올라간다.

## 2. 이 게임의 정서 — 아트가 하게 될 이야기

이 프로젝트의 강점은 정서입니다. 아트는 이 정서를 화면에 옮기는 일이 됩니다.

- **황금빛의 아름다움과 섬뜩함** — 도시 전체가 황금 결정에 뒤덮여 정지해 있다. 사람들은 죽은 게 아니라 굳은 채 보존되어 있다. 아름다워서 더 무섭다.
- **궁핍하지만 담담한 지하** — 지하 마을 칼리고 사람들은 떠밀린 게 아니라 스스로 내려온 사람들이라, 가난해도 비참하지 않다. 투박한 기계와 따뜻한 작업등의 세계.
- **태엽과 기계의 온도** — 주인공 티크는 황동색 로봇(하트 기어 코어, 청록색 눈). 기계의 차가움과 제작자의 손길이 남긴 따뜻함이 공존한다.
- **수직의 세계** — 도시 옥상에서 지하 심층 습지까지 40층 단면. 위로 갈수록 황금빛 정적, 아래로 갈수록 오수와 진흙.

## 3. 지금 바로 볼 수 있는 것

말이 아니라 물건으로 보여드릴 수 있습니다.

1. **플레이 가능한 Windows 빌드** — 오프닝 시퀀스(각성 → 쥐떼 전투 → 쓰러짐 → 공방에서 수리 → 마을)가 실제로 돌아갑니다. 전투·세이브·룸 전환·연출까지.
2. **환경 콘셉트 12장** — `art/concepts/act1/` (칼리고, 정수장치, 지하심층, 터미너스 등 ACT1 지역별)
3. **인터랙티브 월드맵** — `maps/` HTML 뷰어, ACT1 49룸 + 전체 세계 단면
4. **환경 아트 바이블** — `docs/art/act1-environment-art-bible.md` 지역별 콘셉트·전이 규칙

## 4. 확정된 작업 스펙 — 바로 그릴 수 있습니다

아트가 합류하자마자 헤매지 않도록 기술 기준을 먼저 잠가뒀습니다 (`docs/art/production-visual-metrics-v1.md`):

| 항목 | 값 |
|---|---|
| 기준 해상도 | **640 x 360** (720p/1080p 정수 배율) |
| PPU | **64** |
| 카메라 | 10 x 5.625 월드 유닛 |
| 티크 가시 크기 | **42~48 px** |
| 캐릭터 스타일 | 소프트 셰이딩 소형 스프라이트 (플랫 팔레트 픽셀아트가 아니라 할로우나이트 방향) |
| 환경 스타일 | 픽셀 기반 타일·레이어 (포인트 필터) |

주인공 티크는 승인된 액션 프레임 54장(대기·이동·점프·대시·공격 3종)이 이미 있고, 비율·피벗·팔레트 고정 조건이 문서화되어 있습니다 (`docs/art/tique-character-model-lock.md`).

## 5. 합류하면 만들게 될 것

현재 게임은 **그레이박스 + 콘셉트 벤치마크** 상태입니다. 즉 아트가 채울 자리가 명확하게 비어 있습니다:

- **환경** — ACT1 지역별 타일셋·배경 레이어(파랄랙스): 림부스 고철 평원, 칼리고 마을, 정수장치, 지하심층, 터미너스
- **몬스터** — 쥐떼, 하운드, 보스 3종(거대 쥐 렌토, 지하 사냥꾼, 거대 거미)
- **NPC** — 모르비(지하 공학자)와 칼리고 주민들
- **연출 소품** — 오수 폭포, 황금 결정화 오브젝트, 수리 작업대, 기억 코어
- 이펙트·UI는 협의 범위

**첫 과제도 이미 정의되어 있습니다**: 칼리고 마을 룸 하나를 위 스펙으로 완성하는 벤치마크 작업 — 작게 시작해서 스타일을 함께 확정하는 단계입니다.

## 6. 작업 방식

- 기획 문서·맵·룰북이 전부 저장소에 정리되어 있고, 확정/제안이 구분 관리됩니다 — "그때그때 말이 바뀌는" 프로젝트가 아닙니다.
- 빌드·검증 파이프라인이 자동화되어 있어, 아트를 넣으면 곧바로 게임 안에서 확인할 수 있습니다.
- 아트 방향의 최종 결정권은 함께 만드는 스타일 벤치마크를 기준으로 협의합니다. 기존 콘셉트는 후보이지 강제가 아닙니다.

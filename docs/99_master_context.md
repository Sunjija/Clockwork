## 1. 기본 정보

- **제목:** 클락워크 (CLOCKWORK)
- **장르:** 2D 횡스크롤 픽셀아트 메트로베니아
- **플랫폼:** Steam (PC)
- **분량:** 본편 약 10시간(ACT1·2·3), 졸업작품 범위 = ACT1(지하) 약 3시간
- **팀:** 기획·스토리·연출 1인 + 유니티 개발 1인 (+ 아트 담당 합류 예정)
- **핵심 정서:** 시간이 멈춘 도시의 정적, 황금빛으로 굳은 사람들의 아름다움과 섬뜩함, 궁핍하지만 담담한 지하, 태엽과 기계의 따뜻함과 차가움의 공존
- **핵심 원칙:** 절차생성 없음, 영구사망 없음(맵은 손수 제작·영속). 무기·개조의 자유 조합으로 빌드 다양성을 준다.

## 2. 세계관·연표

**설정 핵심:**
- 환경 황폐화로 식물 성장 1/4로 둔화 → 기근 위기
- 크로노스 타워가 물질 '렌티움'을 방출 → 실제로는 인간 인지 속도를 1/4로 늦춤(RUL-002). 렌티움은 자연 물질이 아니라 엘리아스가 발명한 것(정확한 정체는 정의하지 않음)
- 타워는 렌티움을 두 경로로 공급: ①대기 방출(인간 인지 조절) ②폐쇄 배관망(기계 동력 공급, 20년 전 도입). 두 경로는 독립적으로 작동
- 지하 오수는 정제 렌티움이 아니라 '폐렌티움'(사용 후 불안정해진 찌꺼기). 정제 렌티움=기계에 안전한 연료, 폐렌티움=생물에게 위험·기계에게도 불안정
- 폐렌티움은 배관망 회수 경로로 타워에 집결한 뒤 **타워 직하로 배출**된다. 지하의 오수 낙하 기둥(RGN-008)은 이 배출로의 지하 구간 — 타워 직하에서 교차로 대공동으로 쏟아지는 수직 기둥 + 대공동 저수 → 배수 수직갱 → 지하심층으로 이어지는 유로 전체 (v5.5 B-1). 배출은 상시 있었으나 소량이었고, 결정화 후 폭증해 현재의 폭포 규모가 됨 (E-2)
- **반전(진실):** 대중은 타워가 "시간을 가속한다"고 믿지만 실제로는 인지를 둔화시키는 것. 엘리아스와 말파스만 진실을 앎. 엘리아스가 은폐(RUL-003)
- 인지 1/4 → 체감 중력 4배(중력 자체는 아님, RUL-004)
- 인지속도 3단 구배: 중심(1/4) / 변두리(1/3~1/2) / 외곽(1, 정상). 시내/변두리 경계는 행정 구분, 인지 필드는 타워 중심 방사형 — 두 경계는 정확히 일치하지 않음 (E-7)
- **티크의 인지속도도 1/4** — 지상에서 사람들과 어우러져 살도록 설계된 휴머노이드라 도시 표준 템포와 동기화되어 있다. 렌티움의 생물학적 영향은 받지 않으나 인지 템포 자체가 설계상 1/4 (E-1). 지하(무영향 지대) 사람들과의 템포 차는 연출로 표현하지 않음(설정·로그로만 존재)이 기본안
- **결정화:** 황금색 물질. 말파스가 의도적으로 일으킴. 도시 전역. 사람들은 죽지 않고 보존 상태. 부수효과로 렌티움 대량 방출 → 지하 오수 폭증 → 변이 생물(렌토 등) 출현(RUL-006, RUL-011)
- 지하 생물 변이 강도 = 오수 노출 깊이 + 농도/시점. 결정화 이전 점진 변이(하운드=식량 수준) vs 결정화 후 급성 강력 변이(렌토)
- 말파스는 딸을 잃은 후 전신 기계화(생물학적 영향 무관, 정제 렌티움은 동력으로 흡수 가능, RUL-007)
- ACT2 기계 몬스터의 변질 동기: 말파스의 의도적 왜곡(현행) vs **방치설 [제안 — E-5]**(클레아 연구에 출력을 집중하며 배관망 말단 정제도가 방치 속에 저하 — "말파스가 버린 것들"의 정서). 채택 여부 미결
- 안드레스(지하 도시) = 지하 20~25층. 지표 아래 B1~B19는 도시의 **기초 구조층(솔리드)** [제안 — C-1]. 기초층 관통로는 셋뿐: ①폐기 슈트(하행 전용) ②오수 배출로(진입 불가) ③터미너스(유일한 등반로)

**연표:**
- 30년 전: 1차 분쟁(타워 건설 찬반). 반대파(모르비)가 지하로 망명. 엘리아스(40대)·말파스(20대, 제자)가 타워 설계
- 건설 10년(지하 기준)/15년(지상, 프로토타입 렌티움 영향) → 완공 = 20년 전
- 20년 운영
- 2년 전: 2차 분쟁('시간 가속' 이견). 강경파(대표 비바츠) vs 온건파(엘리아스·말파스). 말파스의 딸 클레아(16세)가 시계탑에서 추락→혼수
- 강경파가 탑 점거하나 가속 권한 없음(엘리아스만 보유, 변두리로 도피해 못 찾음)
- 점거 +6개월: 말파스가 습격 → 결정화 발동 → 도시 전역 정지
- 결정화 직후: 렌티움 대량 방출 → 오수 폭증 → 렌토 등 신종 변이 출현. 동시에 티크가 도움 청하러 타워에 올랐다가 말파스에게 내던져짐 → 파손 → 청소 로봇이 고철로 오인해 **시내 상업지구(시장 거리 하부)의 폐기 슈트**로 투하 → 슈트 종점 림부스에 폐기 (C-2). 파손 시 흩어진 기억코어 조각 일부는 오수 유로에 휩쓸려 지하로 운반됨 [제안 — E-8]
- 현재: **티크가 림부스(폐기 지점)에서 각성** → 다리에서 쥐떼 튜토리얼 → 파손 누적으로 쓰러짐 → 모르비가 주워 칼리고서 수리 → ACT1 개시 (v5.5 A-1)

## 4. 칼리고 구역 정의

30년 전 시계탑 건설 반대파가 자발적으로 지하에 내려와 만든 마을. 떠밀린 게 아니라 선택한 것이라 궁핍해도 담담하게 살아감.

**생존 경제 3축:**
1. 폐기계 재활용 (림부스에서 주움)
2. 정화수 (렌티움 오수를 정수 장치로 걸러 식수 확보)
3. 변이 생물 사냥 (하운드=오수에 변형된 개 형태, 식량)

**현재 위협:** 렌토가 정수장치 내부에 둥지를 틀어 시설 가동이 정지 → 배수 중단·물이 끊김 (B-1: 몸으로 관을 막은 게 아니라 시설 정지) → 모르비가 티크에게 도움을 요청하는 동기.

## 5. 월드 구조 (3그룹 × 3지역)

**방위 (C-3, 지도가 정본):** 동측 = 시장 거리·폐기물 투입구·림부스(수거관 종점) / 서측 = 화물 터미널·터미너스, 변두리 끝 엘리아스 은신처 / 지하 서측 = 칼리고·정수장치.

**그룹 A · 지하 안드레스 (ACT1, 졸업작품 범위):**
지상 0F 기준 B20~B25.
- 지역1·시작권: 림부스-칼리고 다리(B20~21, 오수 폭포 횡단), 칼리고(B21), 림부스(B20)
- 지역2·중간권: 정수장치(B23), 교차로(B23), 정수장치 상부/갈고리 시험 공간(B23)
- 지역3·종착권: 지하심층(B25, 최하층 — 렌토 격파 전 침수, 격파 후 수위 하강한 진흙 습지. **수영 조작 없음**, E-3), 터미너스(B22~23, 수직 원통 갱도·ACT1 종착 출구, 상단은 서부 내벽 화물 승강장 → 화물 터미널)
- 오수 낙하 기둥(RGN-008): 지역이 아니라 수직 유로 — 타워 직하 → 대공동 저수 → 배수 수직갱 → 지하심층 (B-1)

**그룹 B · 지상 메티아 (ACT2, 설계 예정):**
시내(중심, 인지 1/4) / 변두리(엘리아스 위치, 인지 1/2) / 외곽(보호벽 바깥, 인지 정상·황폐). 화물 터미널 = CP·저장만 있는 전초기지, 제작·수리는 칼리고 유지 (E-6)

**그룹 C · 크로노스 타워 (ACT3, 설계 예정):**
하층부 / 중층부 / 최상층(옥상, 말파스 최종전)

## 6. ACT1 확정 동선

```
림부스(각성·첫 MOD 부품) → 다리(쥐떼 튜토리얼 → 쓰러짐) → 칼리고(모르비 수리·부품 판명)
→ 교차로 1차 통과(지하심층 문 목격 — 잠김·침수, 렌토가 정수장치를 정지시켜서)
→ 정수장치·렌토 격파 → 티크가 제어반 재가동(배수 재개) → 갈고리(ABL-002) 획득
→ 갈고리 시험 공간(RGN-015)에서 기억①(정체성) 회수
→ 교차로 2차 통과(수위 하강 → 문 열림)
→ 지하심층·지하 사냥꾼 격파 → 더블 점프(ABL-004) 획득 + 기억②(결정화 목격) 회수
→ 터미너스·거대 거미(갈고리+더블 점프 둘 다 필요) → 지상 진출(ACT2)
```

**핵심 설계 원칙:** "가야 할 입구를 먼저 보여주고, 나중에 능력을 얻으면 자연스럽게 갈 수 있게" — 자물쇠를 열쇠보다 먼저 보여준다. (예: 교차로 대공동 1차 통과 시 지하심층 문GAT-101이 잠긴 채 목격되고, 렌토 격파·배수 재개 후에야 열림)

## 7. 몬스터·보스 카드 (ACT1)

| 이름 | 구분 | 위치 | 특성 |
|---|---|---|---|
| 쥐떼 | 잡몹(튜토리얼) | 다리 | 전투 튜토리얼. 한두 마리씩 등장 |
| 하운드 | 잡몹(식량) | 지하 전반 | 오수 변이 개 형태, 칼리고 사람들 사냥감 |
| 렌토 (Lento) | 중간보스/능력 보스 | 정수장치 | 결정화 후 오수 폭증이 낳은 거대 쥐. 정수장치 점거. 격파 시 갈고리 획득 |
| 지하 사냥꾼 | 보스 | 지하심층 | 곰충·갑각류 매복형. 단일 전투(보스방 앞은 환경 예고만). 격파 시 더블 점프+기억② |
| 거미떼 | 잡몹(부하) | 터미너스 | 결정화 이전부터 서식한 점진 변이 선주민(핵 없음, E-4) |
| 거대 거미 | 보스(ACT1 최종) | 터미너스 | 그중 한 개체가 결정화 후 급성 거대화(핵 보유, E-4). 갈고리+더블 점프 둘 다 필요. 기억 없음(순수 게임플레이 클라이맥스) |
| 말파스 | 최종보스급 | 크로노스 타워 옥상 | ACT3. 전신 기계화 |

**ACT별 보스-기억 원칙:**
- ACT1(지하, 생물): 보스는 순수 "길막이". 행동 패턴이 기억 내용과 무관.
- ACT2(지상, 기계, 설계 예정): 기계 몬스터에 결합된 기억코어의 내용 자체가 패턴·기믹·디자인의 원천이 됨(예: "유대" 기억 → 들개형 몬스터가 하울링으로 개체 호출).

## 8. 기억 회수 시스템

**메커니즘 (3겹, 떠먹이지 않고 플레이어가 스스로 잇게):**
1. **섬광 컷신** (최소): 코어 획득 순간 1~2초 단편 이미지, 설명·대사 없음
2. **환경 서사** (주력): 컷신 파편을 다음 구간의 맵·오브젝트가 조용히 확증
3. **텍스트 로그** (양념): 티크의 건조한 기록, 짧고 사실만. 감정은 로그가 말하지 않음(건조함↔플레이어 감정의 간극이 게임 정서)

**기억① — 정체성** (렌토 격파 후, 갈고리 시험 공간에서 회수)
- 컷신: 공방에서 손이 어린 로봇 가슴을 닫아주는 장면(따뜻함) + 끝자락에 황금빛·높은 곳에서 내려다보는 시점이 0.3초 스침
- 환경: 칼리고의 투박한 지하 기술 ↔ 티크의 정교한 지상 제작 대비("나는 여기 출신이 아니다")
- 로그: `기억 코어 #1 동기화. 식별: 제작자 1인. 관계: 미정의.`

**기억② — 결정화 목격** (지하 사냥꾼 격파 후)
- 컷신: 화면 암전 → 도시 전체가 황금빛에 뒤덮이는 광경 0.5~1초
- 환경: 지하심층 진흙 속 정체불명의 황금 결정 파편들
- 로그: `성분 일치: 이전 기록 데이터와 동일 파장.`
- 기억①의 황금빛 위화감이 여기서 회수됨. 내려다보는 시점은 ACT2·3(말파스 연결)에서 회수 예정

**기억코어 위치의 물리 근거 [제안 — E-8]:** 티크 파손 시 흩어진 조각이 오수 유로에 휩쓸려 운반됨 — 상부에 걸린 조각(기억①), 심층까지 침전된 조각(기억②). 회수 순서 = 낙하·침전 순서.

## 9. 게임 시스템 — 능력·무기·개조 삼원 체계

**이동 능력(ABL) = 영구, 보스 보상, 기억 복원과 별개 축**

| ID | 능력 | 해금 |
|---|---|---|
| ABL-002 | 갈고리 사출 | 렌토 격파 |
| ABL-004 | 더블 점프 | 지하 사냥꾼 격파 |
| ABL-003 | 벽타기 | 보류(설계 미정, ACT1 제외) |

**무기(WPN) — 로드아웃 시스템 (2026-07-20 확정)**
- 무기 풀은 본편에서 계속 확장한다. **장착은 동시 3슬롯 고정** — 어떤 3개를 담는가가 빌드를 정의.
- **맨주먹은 항상 사용 가능** (실질 공격 동사 = 주먹 + 장착 3종). 단, **주먹 강화는 로드아웃 슬롯 1개를 차지**한다 — "무기 2 + 강화 주먹" 같은 주먹 빌드 아키타입이 성립.
- 초기 3종: WPN-001 망치(느림·고데미지·묵직) / WPN-002 단검(빠름·저데미지·스타일리시) / WPN-003 대검(중간·중간·균형)
- 전투 중에는 장착된 무기 간 자유 교체(액션 계층). 로드아웃 구성 변경(전략 계층)은 **벤치에서만** (F-4 확정). 빈 슬롯이 있으면 새 무기 획득 시 그 자리에서 자동 장착.
- 공통 액션: 콤보 어택. (가드·패링은 보류 — v5.5 A-2, 방어 액션의 형태는 전투 시스템 회의에서 일괄 결정)
- 콤보는 무기 전환 쌍으로 정의되므로 **로드아웃 선택 = 사용 가능한 콤보 그래프 선택**. ACT1 주먹 축 8종은 어떤 로드아웃에서도 보장.

**콤보 매트릭스 — 핵심 8종 (주먹 축, ACT1·슬라이스 구현 대상)**

| 전환 | 효과 |
|---|---|
| 주먹→망치 | 3연격 후 내려찍기, 스턴 |
| 망치→주먹 | 줄 감아 던지고 위치로 이동(유틸) |
| 주먹→단검 | 6연격, 출혈 부여 |
| 단검→주먹 | 단검 던지고 회수 시 대량 출혈 |
| 주먹→대검 | 빠른 3연격(대검 데미지) |
| 대검→주먹 | 내려베기 + 공격력 버프 |

**무기 간 6종은 본편 확장(ACT1 범위 밖, 보류):** 망치↔단검, 망치↔대검, 단검↔대검

**밸런스 원칙:** 특정 무기가 진행을 "불가능하게 막는 것"은 금지. "더 유리하게"는 권장.

**개조(MOD) = 무기 강화**
- **장착 무기당 1개씩 귀속** (전역 슬롯 아님 — 2026-07-20 확정, §8-4 "동시 3개"의 해석). 무기 3 + 각자의 MOD 1 = 로드아웃의 정체성.
- 획득 경로 (v5.5 A-3으로 일원화): **보물방 3택1** — 결정화 잔해 속 서로 맞물린 부품 3개 중 하나만 뽑을 수 있음(나머지는 뽑는 충격에 깨짐 → 재추첨 없음의 물리적 근거). 배치: 림부스·정수장치 취수구·터미너스.
- 첫 보물방(림부스)은 **정체불명 부품**으로만 획득 — 정체는 칼리고 공방 수리 장면에서 모르비가 판명·장착 (MOD 튜토리얼의 스토리 통합, "인간은 장착 불가·티크만 가능" 전달, B-3)
- 오염 구역(오수 기둥 등)은 생물 진입 불가 지대로 세계관·연출 요소로만 기능 (A-3 — 채집 콘텐츠 없음)
- 게임플레이: 표준 체크포인트 리스폰. (세계관상 "티크가 작동 종료되고 새 개체가 만들어진다"는 사실은 보존, 후반 서사 떡밥으로 활용 가능)

## 10. 진행 중 — 아직 미확정 (다음에 이어서 정할 것)

- **회복 시스템:** 체력=칸형(하트/기어 칸) 확정. 회복 방식=적 처치 시 자원 충전 후 홀드로 소모(할로우나이트식) 방향. 남은 것: ACT별 회복 효율 차등(ACT1=폐렌티움 흡수·낮은 효율, ACT2=정제 렌티움 흡수·높은 효율)의 구체 수치, 홀드 시전 세부(시간·중단 조건)
- **방어 액션:** 가드+패링 / 회피 중심 / 패링 단독 — 콤보 매트릭스와의 상호작용 포함해 전투 시스템 회의에서 일괄 결정 (A-2)
- **체크포인트:** 정체(벤치/충전소 등), 배치, 죽었을 때 정확히 뭘 잃는지 — 미정 (위치만 확정: 보스 직전 + 코너당 1개)
- **재화 시스템:** 잡몹 처치 보상, 범용 화폐 여부 — 미정
- **거대 거미·지하 사냥꾼 구체 패턴:** 컨셉만 있고 패턴 목록 없음
- **ACT1 시나리오 후반부 산문:** 교차로 2차~지상 진출까지 미작성 (렌토전·기억①까지만 작성됨)
- **ACT2·3 내부 설계:** 3지역 틀만 있고 구체 내용 없음
- **캐릭터 정본 문서:** 인물 카드가 임시 정본, 성격·말투 등 상세 자료 없음
- **E-5(방치설)·C-1(기초 구조층)·E-8(기억코어 산포):** v5.5 신설 제안, 채택 결정 필요

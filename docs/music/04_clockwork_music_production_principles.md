# CLOCKWORK 음악 제작 원칙

> 상태: 제작 기준 초안. 지역·서사·진행에 관한 내용은 기획 문서를 따르며, 선율·박자·악기 선택과 Suno 생성 결과는 사용자가 승인하기 전까지 제안 또는 후보로 취급한다.
>
> 문서 역할: 여러 지역과 실험에 반복 적용할 공통 기준이다. 지역별 방향은 [01_region_music_direction.md](01_region_music_direction.md), 실제 생성 및 기각 이력은 [02_suno_anchor_experiments.md](02_suno_anchor_experiments.md), ACT1 작업 순서는 [03_act1_suno_production_plan.md](03_act1_suno_production_plan.md)를 따른다.

## 1. 목표

CLOCKWORK의 음악은 지역을 장식하는 배경음이 아니라 다음 정보를 플레이어에게 전달해야 한다.

- 지금 있는 장소의 기능과 상태
- 티크의 현재 신체·기억 상태
- 탐색, 안전, 위협, 퍼즐, 보스전 중 현재 플레이 역할
- 이전 지역과 다음 지역의 관계
- 기억 회수와 진행에 따라 무엇이 복구되었는지

## 2. 핵심 제작 원칙

### 2-1. 장르보다 플레이 경험을 먼저 정한다

모든 곡은 작곡 전에 아래 한 문장을 확정한다.

`[장소]에서 [상태의 티크]가 [행동]할 때, 플레이어가 [감정 또는 정보]를 느껴야 한다.`

장르명과 악기 목록은 이 문장을 소리로 구현하기 위한 수단으로만 사용한다.

### 2-2. 지역마다 하나의 대표 신호를 둔다

각 지역에는 처음 10~20초 안에 인식 가능한 대표 신호 하나를 둔다.

- 짧은 선율
- 리듬 셀
- 대표 악기
- 화성 진행

여러 특징을 동시에 강조하지 않는다. 대표 신호를 제외한 요소는 이를 지지해야 한다.

### 2-3. 곡보다 테마 계열을 만든다

새 상황마다 무관한 곡을 만들지 않는다. 하나의 기준 테마에서 다음 변주를 파생한다.

- 기본 지역 탐색
- 위험 또는 추격
- 안전 구역
- 기억 회수
- 보스전
- 보스 격파 후 복구

같은 계열은 선율 윤곽, 리듬, 종지 또는 악기 역할 중 최소 하나를 공유한다.

### 2-4. 티크와 세계의 층을 분리한다

- 티크·기억 층: 첼레스타, 펠트 피아노, 벨 계열 금속음
- 지역·시설 층: 준비된 피아노, 파이프 음색, 저현, 리드 악기, 절제된 금속 타악

티크가 회복될수록 모티프의 음역, 길이, 종지와 편성은 분명해진다. 위협이 커질수록 지역 층이 모티프를 가리거나 끊는다.

공통 모티프의 관계를 보여줄 필요가 있다면 같은 악기를 다시 사용할 수 있다. 감정적 온도, 공간 재질, 사회 상태 또는 플레이 역할이 달라지면 먼저 같은 악기의 연주법, 음역, 밀도와 공간감을 바꾼다. 그래도 기존 음색이 새 감정과 충돌할 때만 악기의 역할이나 편성을 교체한다. 지역 수만큼 억지로 악기를 갈아 끼우거나, 익숙한 악기 조합을 모든 감정에 습관적으로 재사용하지 않는다.

| 권역 | 중심 음향군 후보 | 편성 변경이 필요한 신호 |
| --- | --- | --- |
| ACT1 지하 | 첼레스타, 펠트 피아노, 저현, 리드 악기, 조율된 폐기 설비 금속음 | 폐기·생물·공동체 중 곡의 감정과 기능이 바뀌는데 같은 역할 조합만 반복될 때 |
| ACT2 메티아 | 펠트 피아노 연결축, 지속되는 뮤트 현악, 드문 고음 하모닉스, 절제된 프렌치호른 | 지하의 모티프를 유지하되 정지된 시민 사회와 넓은 지상 공간이 연주법·음역·밀도 변화만으로 충분히 구분되지 않을 때 |
| ACT3 타워 | 정밀한 건반 타악, 합성 금속 공명, 대형 금관과 고음 현악의 기능별 레이어 | 메티아의 정적 관찰에서 타워의 정밀한 기능 수행과 수직 진행으로 플레이 역할이 이동할 때 |

음향 변화는 다음 순서로 적용한다.

1. 같은 악기의 연주법과 어택을 바꾼다.
2. 음역과 보이싱을 바꾼다.
3. 음표 수, 레이어 수와 리듬 밀도를 바꾼다.
4. 공간 크기, 잔향 길이와 전후 거리를 바꾼다.
5. 위 변화만으로 새 감정과 장소가 구분되지 않을 때 악기 역할 또는 일부 편성을 바꾼다.

전체 편성 교체는 마지막 수단이다. 같은 악기를 재사용해 서사적 연결이 더 분명해진다면 음색의 연속성을 우선한다.

### 2-5. 음악과 효과음을 섞지 않는다

기계 시설은 리듬과 음색으로 표현하되 실제 게임 효과음을 BGM에 직접 삽입하지 않는다. 배관, 체인, 펌프, 충돌음은 음악적인 타격음이나 연주 가능한 음색으로 추상화한다.

### 2-6. 반복 플레이를 기준으로 검수한다

지역곡은 단독 감상보다 게임에서 반복 재생될 때의 기능을 우선한다.

- 초반부터 계속 고조하지 않는다.
- 새로운 선율을 과도하게 추가하지 않는다.
- A와 B의 차이는 분명하되 같은 곡으로 인식되어야 한다.
- 시작과 끝은 루프 편집이 가능해야 한다.
- 전투가 없는 구간에서 리듬 밀도가 조작감을 압박하지 않아야 한다.

### 2-7. 좋은 씨앗을 변주하고 무작정 다시 뽑지 않는다

Suno에서는 한 프롬프트당 두 후보를 만든다. 결과를 비교한 뒤 한 번에 하나의 큰 변수만 변경한다.

- 선율과 모티프
- 리듬과 템포
- 악기 역할
- 편곡 밀도
- 공간감과 믹스

선택된 후보의 장점은 유지하고 실패한 항목만 바꾼다. 아티스트명이나 기존 게임명을 프롬프트에 사용하지 않는다.

## 3. 제작 절차

1. 맵과 서사에서 장소의 사실을 추출한다.
2. 곡의 플레이 기능과 한 문장 정체성을 작성한다.
3. 인접 곡에서 이어받을 요소와 새 대표 신호를 정한다.
4. A, B, 복귀 구간의 역할을 짧게 설계한다.
5. Suno에서 동일 프롬프트로 후보 두 개를 생성한다.
6. 첫 30초의 기억성, 반복 피로도, 지역 적합성, 구간 변화와 루프 가능성을 평가한다.
7. 선택된 후보에서 한 변수만 바꾸어 다음 두 후보를 만든다.
8. 승인된 곡만 기준 테마 또는 정본 후보로 기록한다.
9. 최종 음원은 게임용 루프, 음량, 페이드와 전환 지점을 별도로 검수한다.

Suno 프롬프트에서는 정확한 마디 수, 음정 규칙과 악기 진입 시점을 DAW처럼 고정하지 않는다. 목표 시간이 중요하면 박자와 BPM에 맞춰 대략적인 길이를 먼저 검산하고, 생성 지시는 도입·중간·복귀·아웃트로의 시간 비율과 가청 결과를 중심으로 작성한다. 세부 편집이 필요하면 승인된 생성물을 기준으로 편집 단계에서 처리한다.

## 4. ACT1 테마 계열

ACT1은 티크의 짧은 기억 모티프를 공통 재료로 사용한다.

| 곡 | 모티프 처리 | 지역 신호 |
| --- | --- | --- |
| 림부스 | 짧고 미완성된 원형 | 절제된 운반 설비 펄스 |
| 림부스-칼리고 교량 | 음 수와 편성을 줄인 이동 변주 | 긴 쉼과 무거운 보행감 |
| 칼리고 | 따뜻한 화성과 인간적인 박자로 변형 | 리드 오르간과 손으로 연주한 금속음 |
| 교차로 대공동 | 저음이나 반주에 숨김 | 여러 방향으로 갈라지는 반복 구조 |
| 정수장치 | 압력 주기에 맞춰 분절 | 파이프 계열의 조율된 타격음 |
| 기억 회수 | 가장 단순한 편성으로 노출 | 장소 기능에 맞춘 응답 악기 |
| 렌토 | 보스 모티프가 티크 모티프를 중단 | 공격과 빈틈을 만드는 압력 리듬 |
| 격파 후 | 일부 종지와 리듬을 복구 | 다시 흐르는 시설의 안정된 펄스 |

## 5. 림부스 기준

### 한 문장 정체성

외곽 지하 폐기물 수용 구역에서 심하게 파손된 티크가 깨어나 탐색할 때, 플레이어는 비극보다 조용한 호기심과 살아 움직이려는 의지를 느껴야 한다.

### 대표 신호

- 첫 10초 안에 등장하는 짧은 다섯 음 모티프
- 느린 6/8박자
- 첼레스타와 펠트 피아노
- 절제된 컨베이어 계열 펄스

### 구조

- A: 모티프를 단순하게 제시
- B: 같은 모티프를 기계적 대위로 확장
- 복귀: 새 선율을 추가하지 않고 A를 더 얇게 재현

### 금지 방향

- 티크를 처음부터 비극적 인물로 단정하는 애가
- 멜로디 변화가 지나치게 많은 판타지 모음곡
- 공포 드론이나 산업 소음 중심의 배경
- 긴박한 추격 리듬
- 실제 기계·충돌 효과음

## 6. 평가 기준

각 후보를 5점 척도로 평가한다.

| 항목 | 질문 |
| --- | --- |
| 기억성 | 첫 30초의 모티프를 다시 알아볼 수 있는가 |
| 지역성 | 다른 지하 지역이 아니라 림부스로 들리는가 |
| 서사성 | 파손, 호기심, 생존 의지가 과장 없이 공존하는가 |
| 반복성 | 10분 이상 반복되어도 피로가 적은가 |
| 연결성 | 교량, 칼리고, 기억 회수와 변주 관계를 만들 수 있는가 |
| 편집성 | 루프와 레이어 전환 지점을 만들 수 있는가 |
